This project implements a large-scale image deduplication system that combines deep learning feature extraction with efficient similarity search algorithms to detect and remove duplicate or near-duplicate images. It integrates FAISS, SimHash LSH (C++), and MinHash with benchmarking of accuracy, speed, and memory usage.

  • Feature Extraction: ResNet50, EfficientNet-B0, ViT
  • Similarity Search: FAISS, SimHash LSH (C++), MinHash LSH
  • Performance Comparison: Accuracy, speed, and memory
  • Deduplication: Representative image selection from clusters

ğŸ§  Implemented Methods

  • FAISS: Exact & approximate nearest neighbor search, GPU acceleration
  • SimHash (C++): Locality-Sensitive Hashing with multi-probing via Pybind11
  • MinHash: Jaccard similarity estimation for baseline comparison
