Image Deduplication Project

A modular pipeline for large-scale image deduplication with FAISS, SimHash (C++), MinHash and a reproducible evaluation framework.

Source

Quick start

Run the pipeline locally (extract features -> build index -> search -> evaluate):

python run_pipeline.py --dataset demo_dataset --extractor resnet --method faiss --build-index

Or load precomputed features:

python run_pipeline.py --dataset demo_dataset --load-features out/features.npy --method simhash

What’s in this repo

  • src/ — core code: extractors, similarity_search, evaluator
  • run_pipeline.py — orchestrator used in experiments
  • demo_dataset/ — example images for quick tests
  • report/ — figures & LaTeX for final report

Evaluation & metrics

The pipeline collects:

  • Precision/Recall/mAP (against ground truth)
  • Index build time & query latency
  • Peak memory (RSS) measured with psutil

How to reproduce

  1. Install dependencies: pip install -r requirements.txt
  2. Build C++ module: cd src/lsh_cpp_module && python setup.py build_ext --inplace
  3. Run pipeline: see Quick start above
